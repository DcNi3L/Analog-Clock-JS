# 3D Animated Clock

This project is a 3D animated clock created using `three.js`, a popular 3D graphics library for JavaScript. The clock is fully interactive, allowing you to rotate and zoom to view it from different angles.

## Features

- **Real-time 3D Animation**: The clock hands move in real-time to display the current time.
- **Interactive Controls**: You can use your mouse to rotate and zoom the camera to view the clock from any angle.
- **Customizable**: The clock's appearance can be easily customized by modifying the `three.js` code.
- **Responsive Design**: The clock will resize to fit any screen size.

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/3d-animated-clock.git
   ```
2. Open the `index.html` file in your web browser.

## Technologies Used

- **`three.js`**: A 3D graphics library for JavaScript.
- **HTML**: The standard markup language for creating web pages.
- **CSS**: The language used to style the visual presentation of web pages.
- **JavaScript**: The programming language that drives the clock's functionality.
